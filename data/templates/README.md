# Data Templates

This directory stores small, version-controlled empirical template data used by
the simulator itself.

Generated DLCoalSim datasets, shards, logs, and scratch data should not be stored
here. They remain ignored by the repository-level `.gitignore`.

Current template packages:

- `human_ne/`: empirical human Ne(t) curves for the
  `empirical_human_ne_template` demography sampler.
